# Descripción
 - Implementación de SOAP para Actividad 1: Ejercicio 1 del TP03

# Instrucciones
 + En directorio SOAP-TP3-FEI: 
  - Instalar dependencias: `npm install` 
  - Ejecutar servicio: `npm start` o `node server.js`

# Aclaraciones
 - Implementado con mysql (testeado con XAMPP)
 - Configuración de acceso a la bd se especifica en el archivo dbConfig.js
 - Esquema de bd especificado en db schema.sql, en root del repo
